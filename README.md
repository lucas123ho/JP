# JSON Page
Make pages with a template and a JSON

## Install
1. Clone this repository
```console
foo@bar:~$ git clone https://github.com/lucas123ho/JP.git
```
2. Inside the repository, install all dependencies
```console
foo@bar:~$ yarn
// or
foo@bar:~$ npm install
```
3. Start the server
```console
foo@bar:~$ yarn start
// or
foo@bar:~$ npm start
```
